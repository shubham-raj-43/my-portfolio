# Shubham Raj — Portfolio 

Modern, responsive personal portfolio website with a futuristic AI/robotic theme (dark UI + neon cyan glow), smooth animations, particle background, and a validated contact form UI.

## Live

After enabling GitHub Pages, your site will be available at:

- `https://<your-username>.github.io/<repo-name>/`

## Features

- Futuristic dark/neon UI (Poppins + Orbitron)
- Particle + animated grid background
- Smooth micro-interactions + hover glow effects
- Scroll progress bar + scroll-reveal animations
- Fully responsive (mobile + desktop)
- Projects modal (View Details)
- Resume download button (`assets/resume.pdf`)
- Footer social icons (LinkedIn, GitHub, YouTube, Email, WhatsApp)
